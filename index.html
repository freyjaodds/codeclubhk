Our mission

The mission of Code Club Hong Kong is to give every child in Hong Kong the chance to learn to code. It is our aim to have Code Club in 20% of primary schools by the end of 2016.

Code Club helps volunteers and schools find each other and creates projects for volunteers to teach at club sessions.

Code Club Hong Kong is part of the Code Club World network.

How it works

We create projects for our volunteers to teach at after school coding clubs. The projects we make teach children how to program by showing them how to make computer games, animations and websites. Volunteers go to their local junior school or other venue, such as a library, for an hour a week and teach one project a week.
